# Frame

Use `frame` to print in frames!

## Example

```go
package main

import (
  "github.com/ramin0/frame"
)

func main() {
  frame.Print("Hello, world!")
}
```

```sh
$ go run main.go
#################
# Hello, world! #
#################
```

## Install

```sh
$ go get -u -v github.com/ramin0/frame
```
